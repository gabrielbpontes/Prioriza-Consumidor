# Prioriza Consumidor

## Sobre o projeto

O **Prioriza Consumidor** é um projeto desenvolvido no contexto da disciplina de Modelagem de Dados Não Estruturados, com foco na aplicação de técnicas de **Processamento de Linguagem Natural (PLN)** para análise de reclamações de consumidores.

O projeto utiliza dados da plataforma **Consumidor.gov.br**, que permite a comunicação direta entre consumidores e empresas, com o objetivo de extrair informações relevantes dos textos das reclamações e apoiar a tomada de decisão.

## Objetivo

O principal objetivo do projeto é desenvolver uma solução capaz de:

- Analisar automaticamente o conteúdo textual das reclamações;
- Estimar a probabilidade de resolução de cada caso;
- Identificar e priorizar reclamações com maior urgência ou criticidade;
- Apoiar processos de atendimento mais eficientes e orientados por dados.

## Motivação

Devido ao grande volume de reclamações registradas diariamente, torna-se difícil realizar uma análise manual eficiente. Nesse contexto, o uso de técnicas de PLN permite automatizar esse processo, contribuindo para uma melhor priorização e resolução dos casos.

## Tecnologias utilizadas

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- SpaCy  
- Técnicas de Processamento de Linguagem Natural (PLN)

## Fonte de dados

Os dados utilizados neste projeto são provenientes da plataforma pública:

- Consumidor.gov.br

Esses dados incluem textos de reclamações, respostas das empresas e informações sobre o status de resolução.

## Impacto esperado

O projeto busca contribuir para:

- Melhorar a eficiência na análise de reclamações;
- Auxiliar na priorização de casos mais críticos;
- Apoiar empresas e instituições na tomada de decisão;
- Melhorar a experiência dos consumidores.
